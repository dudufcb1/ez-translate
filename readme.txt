== Installation ==

### Automatic Installation

1. Go to your WordPress admin dashboard
2. Navigate to Plugins ‚Üí Add New
3. Search for "EZ Translate"
4. Click "Install Now" and then "Activate"

### Manual Installation

1. Download the plugin zip file
2. Upload it via Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Activate the plugin

### Getting Started

1. Go to **EZ Translate** in your admin menu
2. Add your target languages
3. Configure SEO settings
4. Start translating your content!

== Frequently Asked Questions ==

= Is EZ Translate compatible with my theme? =

Yes! EZ Translate is designed to work with any properly coded WP theme. It's especially optimized for FSE (Full Site Editing) themes.

= Can I use EZ Translate with other SEO plugins? =

Absolutely! EZ Translate includes compatibility settings that allow you to use it alongside Yoast SEO, RankMath, and other popular SEO plugins.

= Does it work with WooCommerce? =

Yes, EZ Translate fully supports WooCommerce and other popular plugins. You can translate products, categories, and all e-commerce content.

= Is there a limit to the number of languages? =

No limits! Add as many languages as you need for your global audience.
